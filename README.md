# shape-code-generator

A simple interface that allows you to enter in a shape, and click a button that will generate C++ code that can be saved as a .txt file by the user.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/shape-code-generator.git
cd shape-code-generator
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
